# Hoarde

An Online 5E character sheet app. 

Install Meteor:
```
curl https://install.meteor.com/ | sh
```

Clone this codebase:
```
git clone https://github.com/molybdenym/hoarde.git
cd hoarde
```

Install dependencies and run:
```
meteor npm install
meteor run
```


#### dev stack

- meteor
- react
- apollo (graphql)
- css modules
- flow type
- babel (es6)
